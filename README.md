A sleek, fully-functional **Tetris** clone built with modern web technologies. This project features responsive design, real-time score tracking, and an intuitive UI designed with Radix UI primitives and Tailwind CSS.

---

## 🚀 Live Demo

👉 https://tetris-game-krishshyara25s-projects.vercel.app/

## ✨ Features

- **🎮 Smooth Gameplay:** Classic Tetris mechanics with intuitive keyboard controls.
- **📊 Real-time Scoring:** Track your current score and lines cleared as you play.
- **🌑 Dark Mode Support:** Built-in theme switching via `next-themes`.
- **📱 Responsive Design:** Optimized for various screen sizes using Tailwind CSS.
- **🛠️ UI Components:** Leverages a robust suite of accessible components including Dialogs, Badges, and Tooltips.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [Shadcn/UI](https://ui.shadcn.com/) (based on Radix UI)
- **Animations:** [Lucide React](https://lucide.dev/) icons

---

### Prerequisites

- Node.js (Latest LTS recommended)
- npm, yarn, or pnpm

### 🏗️ Project Structure

The project follows a standard Next.js directory structure:

- `/app`: Contains the main page, layout, and global styles.
- `/components`: Reusable UI components (buttons, cards, dialogs, etc.).
- `/hooks`: Custom React hooks for gameplay logic and UI interactions.
- `/lib`: Utility functions and helper classes.
- `tetris-game.tsx`: Core game component managing the Tetris logic.

### 🎮 How to Play

- **Left Arrow**: Move Piece Left
- **Right Arrow**: Move Piece Right
- **Down Arrow**: Soft Drop (Speed up)
- **Up Arrow**: Rotate Piece
- **Space Bar**: Hard Drop
